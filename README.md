# YTG-MVC-Lookups
![YTG Logo](http://ytgi.com/images/ytgi.png)
<p>Yasgar Technology Group, Inc. .NET Core 3.1 Lookups Demo Website Project</p>
<p>This is not an open source project in the respect that we're not expecting any outside updates to this project.
 However, it is free and available for you to learn some techniques from and utilize in your organization.</p>


## What is it?
<p>This is a companion project to the blog post regarding <a href="https://jackyasgar.net/look-up-engine-basics/" target="_blank">Look-up Engine Basics</a>.</p>
<p>It is also an example project to illustrate the importance of proper paging techniques discussed
in the blog post <a href="https://jackyasgar.net/series/ultimate-guide-to-mvc-paging/" target="_blank">Ultimate Guide to MVC Paging</a>.</p>

## Techniques Illustrated

* Lookups Management
* Paging the proper way
  * Using PagedResult and SearchModel DTOs
* AJAX for returning Partial Views
* Asynchronous Controller Methods
* Task / Await Pattern
* Lazy loading of child objects
